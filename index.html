Through my time at App Academy, we were tasked with making three main portfolio projects. My first was the API project, the second a group project called Mock Make, and the third a solo capstone project called Thermos.

Projects in Detail

API project

This project was our first taste of creating a full stack application, starting with the backend Express routes before moving to building the frontend React server. My project is a clone of the site Meetup and all seeder data and images are inspired by Five Nights at Freddy's.

Mock Make

Mock Make was built quickly on the heels of learning Python as a second language. This group project was built as an Etsy clone with cohort-mates Teagan Stutsman and Nur Unlu. It is built with a React frontend server and a Flask backend server.

Thermos

Our last project before completing the App Academy bootcamp was the capstone. For mine, I made a cozy, coffee-colored Tumblr clone called Thermos using React and Flask.
